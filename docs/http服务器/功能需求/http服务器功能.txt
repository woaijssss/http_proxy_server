3、解析请求方法：
	支持GET、POST、PUT和DELETE
4、解析url：
	校验url格式
	支持url获取、uri解析：
	（1）GET方法支持获取各个参数
	（2）POST、PUT方法，不支持带参请求
	（3）PUT方法支持以路径的方式更新指定资源
	（4）DELETE方法支持以路径的方式和参数形式删除指定资源

	校验header格式
	header内容，不做校验，直接透传

	校验body格式
